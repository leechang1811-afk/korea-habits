# Metrics

## North Star
- Weekly Active Users (WAU)

## Primary KPI
- D1 Retention
- D7 Retention
- Habit Completion Rate (daily)
- Average Habits per Active User

## Monetization KPI
- Ad Impression per DAU
- eCPM (platform report)
- Revenue per DAU

## Event Tracking (MVP)
- `app_open`
- `habit_add`
- `habit_toggle`
- `habit_delete`
- `ad_interstitial_shown`
