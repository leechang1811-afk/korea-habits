# Monetization Strategy (Ads First)

## Ad Surfaces
- 하단 배너: 항상 노출 (핵심 인터랙션 가리지 않음)
- 전면 광고: 체크 5회 누적 시 1회 노출

## UX Guardrails
- 사용자가 액션 직후 이탈하지 않도록 과도 노출 금지
- 체크 버튼 클릭 시 매번 광고를 띄우지 않음
- 광고 실패 시 핵심 플로우를 절대 막지 않음

## Revenue Hypothesis
- 무료 사용자는 광고로 수익화
- 재방문이 늘수록 전면 광고 트리거가 자연 증가

- 배너 테마/위치 테스트
