# Evaluation (PO / Designer / User)

## Top 5 Issues
1. 로그인 없이 로컬 저장만 제공되어 기기 변경 시 데이터 유실 가능
2. 습관 카테고리/템플릿이 없어 초보 사용자는 첫 입력이 어려울 수 있음
3. 주간 성과 시각화가 단순 숫자 중심이라 동기부여가 약할 수 있음
4. 광고 타이밍이 일부 사용자에게는 갑작스럽게 느껴질 수 있음
5. 알림 기능이 없어 재방문 리마인드가 약함

## [PO] Product Assessment
- 실제 문제를 해결하나? **Yes** (빠른 체크 니즈 충족)
- KPI 타당한가? **Yes** (리텐션/완료율/광고 수익 중심)
- 과복잡한가? **No** (MVP 범위 적절)

## [Designer] UX Assessment
- 온보딩 없이 바로 시작 가능해 장점
- 빈 상태에서 "어떤 습관을 입력할지" 가이드가 약함
- 체크 버튼 시각 피드백은 명확함

## [Real User] Usage Assessment
- "바로 쓸 수 있어서 좋다"
- "데이터 백업이 없어서 불안하다"
- "주간 리포트가 좀 더 직관적이면 계속 쓸 듯"

## UX Flaws
- 첫 사용자 가이드 부족
- 성취 시각화 약함

## Product Flaws
- 백업/복구 없음
- 알림 없음

## Monetization Flaws
- 전면 광고 빈도 최적화 전
- 세그먼트별 광고 정책 미분화
