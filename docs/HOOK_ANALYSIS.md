# Hook 모델 + 행동/인지심리학 기반 앱 분석

## Hook 모델 (Nir Eyal)

| 단계 | 현재 상태 | 개선 |
|------|----------|------|
| **Trigger** (외부/내부) | 홈에서 "시작하기" | 스트릭 손실 두려움, 오늘의 목표, 변동적 문구 |
| **Action** | 1탭으로 시작 | → 유지 (이미 간단함) |
| **Variable Reward** | 퍼센타일, 콤보, 결과 | 랜덤 칭찬, 예측 불가능한 보상 문구 |
| **Investment** | 스트릭, 리더보드 기록 | 손실 회피 강화, "N단계까지 왔는데" (Zeigarnik) |

## 행동심리학 적용

- **Loss Aversion (손실 회피)**: "스트릭이 끊어지기 전에!" - 오늘 안 하면 N일 연속 날아감
- **Zeigarnik Effect**: 미완료 과제 기억 → "18단계까지 왔는데, 한 번만 더?"
- **Variable Reinforcement**: 스키너 - 예측 불가능한 보상이 습관 강화
- **Endowed Progress**: "이미 5단계 클리어" - 시작한 것처럼 느껴져 계속 유도
- **Commitment & Consistency**: 작은 yes(1회 플레이) → 큰 yes(습관화)

## 인지심리학 적용

- **Curiosity Gap**: "다음 단계 통과하면 상위 N%" - 궁금증 유발
- **Chunking**: 20단계를 5-5-5-5로 인지하거나, 마일스톤(9, 15)으로 구간화
- **Completion Desire**: 프로그레스 바로 "거의 다 왔어요" 체감

## 기본 욕구 (Reiss)

- **Achievement**: 최고 기록, 상위 %, Champion
- **Competition**: 리더보드, 한국인 vs 나
- **Belonging**: 스트릭 "N일 연속" 커뮤니티감
- **Mastery**: 강점/약점, 개선 팁
- **Curiosity**: "다음엔 몇 %?"
- **Autonomy**: 공유, 다시 도전 선택
