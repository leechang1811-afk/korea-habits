# PRD - Korea Habit MVP

## Problem
기존 습관 앱은 기능이 많아 진입 장벽이 높다.  
사용자는 기록보다 "빠른 체크"를 원한다.

## MVP Scope
- 습관 생성/삭제
- 오늘 체크 토글
- 연속일수 표시
- 최근 7일 체크 합계 표시
- 오늘 달성률 표시
- 로컬 저장(localStorage)
- 예외 상황에서도 동작 가능한 기본 에러 바운더리

## Out of Scope
- 소셜 기능
- 클라우드 동기화
- 푸시 알림
- 복잡한 통계/리포트

## User Flow
1. 앱 진입
2. 습관 입력 후 추가
3. 오늘 체크하기
4. 달성률/연속일수 확인
5. 다음날 재방문

## Acceptance Criteria
- 새 습관을 추가하면 리스트 최상단에 노출된다.
- "오늘 체크하기"를 누르면 완료 상태로 바뀐다.
- 완료 상태를 다시 누르면 체크가 해제된다.
- 새로고침 후에도 데이터가 유지된다.
- 습관이 0개일 때 빈 상태 UI가 보인다.
